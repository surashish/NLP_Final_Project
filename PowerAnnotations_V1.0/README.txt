README
------

This folder contains the following files/folders

1. AnnotatedThreads/ - The folder which contains threads annotated with all four types of power considered in this study (See references below for the paper describing the annotations)

2. Manual.pdf - The annotation manual used for obtaining these annotations, including the definitions of power types, special instructions etc.

3. README.txt - This file


References
----------

Prabhakaran, Vinodkumar; Rambow, Owen and Diab, Mona. Annotations for Power Relations on Email Threads. In Proceedings of LREC 2012. Istanbul, Turkey (May, 2012)
